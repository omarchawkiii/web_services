Sentinel NOC Hub Mobile API

Complete REST API reference for integrating the mobile application with the Sentinel NOC Hub. All endpoints return JSON. Authentication is handled via Laravel Sanctum bearer tokens.

Base URL
https://your-hub-domain.com
API Prefix
/api/hub/mobile
Format
JSON (application/json)
Auth
Bearer Token (Sanctum)
🔑
Authentication

How to authenticate requests to the mobile API.

The mobile API uses Laravel Sanctum token-based authentication. After logging in, you receive a plaintext bearer token that must be included in the Authorization header for every protected request.

Request Header (all protected endpoints)
HTTP Header
        Authorization: Bearer {your_token}
        Content-Type: application/json
        Accept: application/json
ℹ️
Only the login endpoint is public. All other endpoints require a valid bearer token. Tokens are single-session — logging in again invalidates previous tokens.
⚠️
Data returned by each endpoint is scoped to the authenticated user's assigned locations. A user will never see data from locations they are not associated with.
⚡
Error Codes

Standard HTTP error responses returned by the API.

Status Meaning Common Cause Response Body
200 OK Success Request processed correctly Endpoint-specific JSON
401 Unauthorized Missing or expired token {"message": "Unauthenticated."}
403 Forbidden Resource not in user's locations {"message": "Forbidden."}
404 Not Found Invalid ID in URL parameter {"message": "Not found."}
422 Validation Error Missing or invalid request fields {"message": "...", "errors": {...}}
500 Server Error Internal server exception {"message": "Server Error."}

Login, profile retrieval, and logout.

POST /api/hub/mobile/login No Auth

Authenticates a user and returns a Sanctum bearer token. Any existing mobile sessions for this user are revoked.

Request Body (JSON)
FieldTypeRequiredDescription

Response 200 OK

Schedules

Cinema screening schedules. All schedule endpoints accept an optional date query parameter (YYYY-MM-DD) — defaults to today when omitted. The day window runs from 5:00 AM to 4:59 AM the following day (not midnight-to-midnight) to match cinema operational shifts.

NOC Sync API — Schedules

Endpoints called by NOC instances to sync schedule data to the hub. Authentication: Authorization: Bearer {noc_instance.api_key}. Always call clear first, then send chunks via sync.

POST /api/noc/schedules/clear NOC API Key

Deletes all schedules for this NOC instance within the given date range. Must be called before sending sync chunks so that sessions deleted in the NOC are also removed from the hub.

Request Body (JSON)

NOC Sync API — Playback Details

Endpoint called by NOC instances to push hardware detail data to the hub. Authentication: Authorization: Bearer {noc_instance.api_key}. This is a server-to-server push — not called by the mobile app.

POST /api/noc/playback-details/sync NOC API Key

Receives server details, projector details, sound details, storage devices, SMART health data, and lamp history for all screens of the NOC instance. All sub-arrays are optional — only include the types that have changed.

⚠️
This endpoint uses upsert logic (updateOrCreate keyed by noc_instance_id + screen_id). Sending data replaces the existing record for that screen.
